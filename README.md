## Overview

The simplicity of the <a href="#">SuperDuper</a> interface is a deliberate design choice that reflects the application's understanding of what matters in backup software. The source and destination selection, the backup mode choice, and the option to set a schedule are presented clearly on a single screen without the complexity that can cause users to make mistakes in tools with more elaborate configuration options. This simplicity reduces the friction of the backup setup process, which is important because backup tools that are complex to configure are tools that users procrastinate about setting up, resulting in periods without protection that create the exact risk that backup is intended to eliminate.

The copy script system in <a href="#">SuperDuper</a> provides the customization capability that different backup scenarios require, allowing the exact content included in or excluded from a backup to be defined precisely. The standard "Backup — all files" script creates a complete bootable clone, while custom scripts can exclude specific large directories that are already backed up through other means — video archives, virtual machine files — to reduce backup time and destination storage requirements. The ability to create different scripts for different backup tasks enables a tiered backup strategy where <a href="#">different backup jobs</a> serve different purposes without requiring multiple separate backup tools.

The scheduling capabilities in <a href="#">SuperDuper</a> address the practical challenge of ensuring that backups actually happen regularly rather than being performed only when the user remembers to initiate them. Scheduled backups run automatically at configured times, with the application waking the system if necessary to ensure the backup completes even when the computer has been put to sleep. The email notification system sends completion reports to a configured address after each scheduled operation, providing confirmation that protection is current without requiring the user to check the application manually. The detailed log maintained for every backup operation provides a complete audit trail of what was copied, what was skipped, and any errors that occurred — essential information for verifying that backups are complete and that protection is functioning as expected. For users who prioritize reliable, understandable protection over feature complexity, <a href="#">SuperDuper OSX</a> provides the most trusted bootable backup experience available.

## Key Features

- After-copy <a href="#">shell script execution</a> for custom post-backup automation tasks
- Email notification reporting <a href="#">backup completion or errors</a> after unattended operations
- Backup to <a href="#">disk image files</a> for storing compressed bootable copies on larger drives
- Multiple destination support creating <a href="#">parallel backups</a> to different drives on a schedule
- Detailed <a href="#">backup log viewer</a> showing every file operation for full audit capability
- Growl and <a href="#">notification center alerts</a> for backup status without requiring app focus
- Pre-backup <a href="#">sleep prevention</a> ensuring completion without system interruption
- Volume <a href="#">selection interface</a> clearly identifying source and destination to prevent errors
- Sparse bundle and <a href="#">sparse disk image</a> creation for space-efficient image backups
- Full compatibility with all <a href="#">Intel and Apple Silicon</a> hardware configurations

## Additional Information

SuperDuper requires macOS 10.15 or later and is compatible with Intel and Apple Silicon hardware. The perpetual license covers all cloning, Smart Update, scheduling, and Sandbox features with no subscription costs. Basic cloning is available free; a license unlocks scheduling and Smart Update.
